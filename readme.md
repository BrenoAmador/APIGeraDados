# APITesteDev

API de teste em ASP.NET Core 8 com JWT e envio de e-mails via SMTP (MailKit).

## Funcionalidades

- Autenticação com JWT
- Controle de usuários e autorização de rotas
- Envio de e-mails com anexos
- Swagger para documentação da API
- Configuração via `appsettings.json` ou variáveis de ambiente
